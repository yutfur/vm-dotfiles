# vm-dotfiles

# Requirement
- Arch Linux on VM Workstation

# Download
```
cd $HOME
git clone https://github.com/yutfur/vm-dotfiles.git
```

# Setup
```
cd $HOME/vm-dotfiles
chmod u+x ./bin/link.sh
./bin/link.sh
```

# Install
```
cd $HOME/vm-dotfiles
chmod u+x ./bin/install.sh
./bin/install.sh
```
